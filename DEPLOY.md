## To go live on Vercel (free, ~5 minutes)

1. Create a GitHub account at github.com (if you don't have one)
2. Create a new repository called `janta-prints`
3. Upload all files from this folder to that repository
4. Go to vercel.com → Sign up with GitHub
5. New Project → Import `janta-prints` repo
6. Click Deploy — done. You get `janta-prints.vercel.app`
7. Add your domain: Settings → Domains → Add `jantaprints.co.ke`
